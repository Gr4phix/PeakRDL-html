VS

Absolute Address:
Base Offset:
0x10

Description

This register indicates the major and minor version of the AHCI specification that the HBA implementation supports.

The upper two bytes represent the major version number, and the lower two bytes represent the minor version number.

Example: Version 3.12 would be represented as 00030102h. Three versions of the specification are valid: 0.95, 1.0, and 1.1.
