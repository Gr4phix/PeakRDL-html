EM_LOC

Absolute Address:
Base Offset:
0x1c

Description

The enclosure management location register identifies the location and size of the enclosure management message buffer.

Contents

Bits Identifier Access Reset Decoded Name
[15:0] SZ r 0x0 Buffer Size
[31:16] OFST r 0x0 Offset

Field Descriptions

SZ - Buffer Size

Specifies the size of the transmit message buffer area in Dwords. If both transmit and receive buffers are supported, then the transmit buffer begins at ABAR[EM_LOC.OFST*4] and the receive buffer directly follows it. If both transmit and receive buffers are supported, both buffers are of the size indicated in the Buffer Size field. A value of '0' is invalid.

OFST - Offset

The offset of the message buffer in Dwords from the beginning of the ABAR.
